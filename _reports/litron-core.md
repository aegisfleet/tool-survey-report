# **LITRON CORE 調査レポート**

## **1. 基本情報**

* **ツール名**: LITRON CORE
* **ツールの読み方**: リトロン コア
* **開発元**: NTTデータ (NTT DATA)
* **公式サイト**: [https://www.nttdata.com/jp/ja/lineup/litron-core/](https://www.nttdata.com/jp/ja/lineup/litron-core/)
* **関連リンク**:
  * 製品紹介資料DL: [https://go.nttdata.com/l/547422/2025-06-30/8ykt9r](https://go.nttdata.com/l/547422/2025-06-30/8ykt9r)
* **カテゴリ**: 自律型AIエージェント
* **概要**: NTTデータが開発したエージェント型AI基盤。人からAIへの業務委託・自動化を実現し、オフィスワーカーの汎用業務の効率化を支援するエンタープライズ向けプラットフォームです。

## **2. 目的と主な利用シーン**

* **解決する課題**:
  * 業務の属人化および反復的な作業による生産性の低下
  * 部門を横断する複雑な業務プロセスの非効率
  * 既存のRPA等では対応しきれない、判断を伴う業務の自動化
* **想定利用者**:
  * 企業の全部門（営業、企画、総務、経理など）のオフィスワーカー
  * DX推進部門
* **利用シーン**:
  * **調査・分析**: マーケット調査や競合分析などの情報収集と整理
  * **資料作成**: 収集した情報に基づいたレポートや提案書のドラフト作成
  * **業務プロセスの自動化**: 自然言語での指示によるワークフローの自動生成と実行

## **3. 主要機能**

* **パーソナルエージェント**: ユーザー履歴をもとに過去の関連情報を検索・選別し、個人の特性や指示に応じたプランニングを実施します。
* **自然言語でのワークフロー生成**: 自然言語で指示するだけで、業務プロセスをAIが自動設計・実行します。作成されたワークフローは再利用可能です。
* **マルチエージェント構造**: 複数のエージェントが連携し、複雑なタスクを分担して遂行します。必要に応じて新規エージェントも自動生成されます。
* **業務特化エージェント**: 特定業務（調査、分析、資料作成など）に特化したエージェントが用意されており、高精度な支援を提供します。
* **SmartAgent**: 状況に応じて適切な判断を行い、自律的にタスクを遂行する機能群です。

## **4. 開始手順・セットアップ**

* **前提条件**:
  * 企業契約が必須（個人向けプランはなし）
  * 導入コンサルティング等が必要になるケースが一般的
* **導入手順**:
  1. 公式サイトの[お問い合わせフォーム](https://www.nttdata.com/jp/ja/contact-us)から問い合わせ
  2. NTTデータの担当者によるヒアリングと提案
  3. 契約・環境構築
  4. 利用開始（オンボーディング支援あり）
* **初期設定**:
  * 企業内のデータ連携やセキュリティポリシーの設定が必要になります。

## **5. 特徴・強み (Pros)**

* **日本企業に最適化された信頼性**: 国内最大手のSIerであるNTTデータが提供するため、セキュリティやサポート体制が強固であり、日本の商習慣にも適合しています。
* **柔軟なマルチエージェント連携**: 単一のAIではなく、複数のエージェントが協調して動くことで、単独では難しい複雑な業務課題を解決できます。
* **ワークフローの資産化**: 一度生成したワークフローを再利用できるため、組織全体でのナレッジ共有と業務スピード向上が期待できます。

## **6. 弱み・注意点 (Cons)**

* **導入ハードルの高さ**: B2Bエンタープライズ製品であるため、個人や小規模チームが手軽に試すこと（PLG的な導入）は困難です。
* **情報の透明性**: 詳細な技術仕様やAPIドキュメント、料金体系がWeb上では公開されておらず、検討には営業担当との接触が必須です。
* **コスト**: エンタープライズ品質である分、導入・維持コストは相応にかかることが予想されます。

## **17. 総評**

* **総合的な評価**:
  * NTTデータが提供するLITRON COREは、セキュリティと信頼性を最優先する日本企業にとって有力な選択肢です。特に「マルチエージェント」構造により、単なるチャットボットを超えた複雑な業務代行が可能である点が革新的です。一方で、導入にはコンサルティングや個別開発が前提となるため、手軽に導入できるSaaS製品とは一線を画します。
* **推奨されるチームやプロジェクト**:
  * 金融、公共、製造など、機密情報の取り扱いが厳格な大企業のDXプロジェクト。
  * 部門を横断する複雑な業務プロセスを自動化したい業務改革チーム。
* **選択時のポイント**:
  * 「Office操作の補助」で十分ならMicrosoft 365 Copilot、「定型作業の自動化」ならUiPathが適していますが、「判断を伴う複雑な業務の委任」と「国内法規制・セキュリティへの完全適合」を求める場合はLITRON COREが最適解となります。
