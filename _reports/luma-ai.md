# **Luma AI 調査レポート**

## **1. 基本情報**

<!--
【ガイドライン】
- 必須: ツール名、開発元、公式サイト、カテゴリ、概要
- 概要は2-3文で簡潔に記述
-->

* **ツール名**: Luma AI
* **ツールの読み方**: ルマエーアイ
* **開発元**: Luma Labs, Inc.
* **公式サイト**: [https://lumalabs.ai/](https://lumalabs.ai/)
* **関連リンク**:
  * ドキュメント: [https://lumalabs.ai/learning-hub](https://lumalabs.ai/learning-hub)
* **カテゴリ**: 動画/メディア
* **概要**: テキストや画像から高品質な動画や3Dコンテンツを生成するAIプラットフォーム。特に動画生成モデル「Dream Machine」が主力製品であり、物理法則に忠実な動きと高解像度な映像生成を実現しています。APIも提供されており、外部アプリケーションへの組み込みが容易です。

## **2. 目的と主な利用シーン**

<!--
【ガイドライン】
- 3-5項目を箇条書きで記述
- 想定利用者を具体的に記載
-->

* **解決する課題**: 専門的なスキルや高価な機材を必要とせず、誰でも簡単に高品質なビデオや3Dコンテンツを作成できるようにする。
* **想定利用者**: メディア、エンターテイメント、マーケティング、広告業界のクリエイター、デザイナー、開発者。
* **利用シーン**:
  * 短編映画やアニメーションの制作
  * 製品のプロモーションビデオ作成
  * SNS向けの動画コンテンツ制作
  * ゲームやAR/VR向けの3Dアセット作成

## **3. 主要機能**

<!--
【ガイドライン】
- 5-10項目の主要機能を箇条書きで記述
- 各機能は1-2文で概要を説明
-->

* **Dream Machine**: テキストや画像から高品質な動画を生成する主力機能。
* **Modify with Instructions**: 自然言語の指示に基づき、生成済みまたはアップロードした動画のオブジェクト削除、スタイル変更、背景修正などを実行する編集機能。
* **Modify Video**: 既存の動画のスタイルや環境、要素を変更する動画編集機能。
* **Reframe**: 画像や動画を任意の方向に拡張・リサイズする機能（アウトペインティング）。
* **3D Capture**: スマートフォンのカメラを使い、現実の物体や風景から3Dモデルを生成する機能。
* **Luma API**: 開発者がLumaのAIモデルを自身のプロダクトに組み込むためのAPI。PythonおよびNode.js向けのSDKが提供されている。

## **4. 開始手順・セットアップ**

<!--
【ガイドライン】
- アカウント作成から「Hello World」的な最初の動作確認までの手順
- 主要なコマンドや設定項目を記載
-->

* **前提条件**:
  * Luma Labsアカウントの作成
  * API利用の場合はAPIキーの取得
  * Python 3.8以上 または Node.js 18以上（SDK利用時）
* **インストール/導入**:
  ```bash
  # Python SDK
  pip install lumaai

  # Node.js SDK
  npm install lumaai
  ```
* **初期設定**:
  * 環境変数 `LUMA_API_KEY` に取得したAPIキーを設定する。
* **クイックスタート**:
  * **Web版**: 公式サイトにログインし、プロンプト入力欄にテキストを入力して「Generate」をクリックするだけで動画生成が開始される。
  * **API (Python例)**:
    ```python
    from lumaai import LumaAI
    client = LumaAI()
    generation = client.generations.create(prompt="A teddy bear running in New York city")
    ```

## **5. 特徴・強み (Pros)**

<!--
【ガイドライン】
- 3-5項目を箇条書きで記述
- 競合との差別化ポイントを明確に
-->

* **高品質な動画生成**: 最新モデルにより、物理法則に沿った自然で一貫性のある動きを持つ動画を生成できる。
* **高度な編集機能**: `Modify with Instructions`や`Reframe`といった高度な編集機能により、生成後のコンテンツを柔軟に調整できる。
* **直感的な操作性**: 専門的な知識がなくても、直感的なUIで高品質なコンテンツを制作できる。
* **開発者向けAPI**: 公式SDKが頻繁にアップデートされており（2026年1月時点でv1.19.1）、外部連携が容易。

## **6. 弱み・注意点 (Cons)**

<!--
【ガイドライン】
- 3-5項目を箇条書きで記述
- 日本語対応の状況は必ず含める
-->

* **複雑な料金体系**: クレジットベースの課金体系が複雑で、機能ごとに消費量が異なるためコストの見積もりが難しい。
* **日本語情報の不足**: 公式サイトやドキュメントは英語が中心で、日本語のサポートや情報が限られている。
* **商用利用の制限**: 無料プランおよび$9.99/月のLiteプランでは商用利用が許可されておらず、ウォーターマークが付与される。
* **音声の非対応**: 動画に音声を付与する機能は限定的である（他社ツールとの併用が推奨される場合が多い）。
