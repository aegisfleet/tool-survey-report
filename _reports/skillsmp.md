# **SkillsMP 調査レポート**

## **1. 基本情報**

* **ツール名**: SkillsMP (Skills Marketplace)
* **ツールの読み方**: スキルズエムピー / スキルズマーケットプレイス
* **開発元**: SkillsMP
* **公式サイト**: [https://skillsmp.com](https://skillsmp.com)
* **関連リンク**:
  * APIドキュメント: [https://skillsmp.com/docs](https://skillsmp.com/docs)
* **カテゴリ**: AIエージェント基盤
* **概要**: SkillsMPは、Claude CodeやOpenAI Codex CLI、ChatGPTなどのAIエージェント向けスキル（Agent Skills）を170万件以上集約した最大のマーケットプレイスプラットフォームです。オープンな標準規格 `SKILL.md` に対応し、AIセマンティック検索、REST API、職業ベースのカテゴリ分類などを提供しています。

## **2. 目的と主な利用シーン**

* **解決する課題**: 世界中に散らばっている膨大な数のエージェントスキルの中から、目的に合った高品質なスキルを探し出すコストを削減し、自社または自身のAIエージェントに素早く組み込むことを可能にする。
* **想定利用者**: Claude CodeやOpenAI Codex CLIなどを利用するソフトウェアエンジニア、データサイエンティスト、システム管理者などの各種プロフェッショナル。
* **利用シーン**:
  * 自分の職業や役割（例：ソフトウェア開発、管理職など）に特化したスキルを発見したい時
  * Claude CodeやCodex CLIに新しい機能（例：特定のAPI操作、PPT生成）をワンコマンドで追加したい時
  * REST APIを通じて、自社の分析ツールやエージェントワークフローにスキルデータを統合したい時

## **3. 主要機能**

* **巨大なスキルライブラリ**: 170万件以上のスキル（1.7M+ SKILL.md）を収録し、あらゆるユースケースをカバー。
* **スキル検索とプレビュー**: AIセマンティック検索やMasonryレイアウトによる詳細な説明プレビュー機能を備え、直感的にスキルを探せる。
* **職業ベースのカテゴリ分類**: 米国労働省のSOC分類に基づく800以上の職業別カテゴリから、自分の職務に関連するスキルをブラウジング可能。
* **REST API**: 外部の検索、分析、または自律型エージェントのワークフローから直接170万以上の公開スキルを検索・取得できるAPI。
* **ワンクリックインストール**: `skills.sh` を利用し、npx/bunx/pnpmなどを通じてコマンドラインから直接スキルをインストール可能。
* **SKILL.md標準対応**: オープンスタンダード `SKILL.md` に準拠したスキルを収集・表示し、マルチエージェントで利用可能。

## **4. 開始手順・セットアップ**

* **前提条件**:
  * Webブラウザ（検索や閲覧用）
  * 対象となるAIエージェント（Claude Code、OpenAI Codex CLIなど）
* **インストール/導入**:
  Webプラットフォームであるためインストールは不要です。スキルを利用する場合は、コマンドラインからワンクリックインストールを使用します。

  ```bash
  # 例: skills.sh を利用したインストール
  npx skills.sh install <skill-name>
  ```

* **初期設定**:
  * 特にアカウント登録なしで検索やブラウジングが可能です。
  * ダウンロードしたスキルの `SKILL.md` の内容を確認し、対象エージェントの設定ファイル（例: `CLAUDE.md`）に統合します。
* **クイックスタート**:
  * [公式サイト](https://skillsmp.com) にアクセスし、検索バーに目的を入力するか、「Explore Occupations」から職業別のスキルを閲覧します。

## **5. 特徴・強み (Pros)**

* **圧倒的な規模**: 170万件（1.7M+）以上という桁違いのスキルが登録されており、ほぼあらゆるユースケース・職業に対応するスキルが見つかる。
* **高い統合性・拡張性**: REST APIを提供しているため、開発者が独自のツールやエージェントシステムにSkillsMPのデータベースを直接組み込める。
* **職業特化のブラウジング**: 米国労働省のSOC分類を用いた職業別のスキル検索が可能で、役割に応じた実用的なツールを発見しやすい。
* **マルチエージェント対応**: Claude Codeだけでなく、OpenAI Codex CLIなど同じ標準（SKILL.md）を採用する複数のエージェントツールで利用可能。

## **6. 弱み・注意点 (Cons)**

* **品質のばらつき**: コミュニティ投稿型のプラットフォームであるため、スキルの品質や安全性は玉石混交であり、導入前の確認が必要。
* **公式サポートの不在**: コミュニティプロジェクトであるため、企業レベルの公式サポートはなく、問題解決はコミュニティやGitHub Issuesに依存する。
* **英語中心**: 多くのスキルやドキュメントは英語で記述されており、日本語のスキルは相対的に少ない。

## **10. エコシステムと連携**

### **10.1 API・外部サービス連携**

* **API**: 170万件のスキルを検索・取得するための**REST API**が提供されており、API経由で自社のツールやエージェントシステムへの直接統合が可能。
* **外部サービス連携**:
  * **GitHub**: 多くのスキルの実際のホスティング元。
  * **Claude Code**: SKILL.mdの主要なターゲットエージェント。
  * **OpenAI Codex CLI / ChatGPT**: SKILL.md互換のAIモデル・エージェント。

### **10.2 技術スタックとの相性**

| 技術スタック | 相性 | メリット・推奨理由 | 懸念点・注意点 |

## **11. セキュリティとコンプライアンス**

* **認証**: 基本的な検索やAPI利用はオープン（詳細なAPIキーなどの要件は公式Docsを参照）。

* **準拠規格**: 公式サイトではセキュリティ規格等の詳細は公開されていない。問い合わせが必要。

## **12. 操作性 (UI/UX) と学習コスト**

* **UI/UX**: モダンでシンプルなインターフェース。ターミナルライクなデザイン要素もあり、開発者にとって馴染みやすい。
* **学習コスト**: 検索してGitHubからクローン/コピーするだけなので、学習コストはほぼゼロ。

## **13. ベストプラクティス**

* **効果的な活用法 (Modern Practices)**:
  * **まずは検索**: 自分でプロンプトを書く前に、同様の機能を持つスキルがないか検索する。
  * **コード確認**: ダウンロードしたスキル（特にスクリプトを含むもの）は、実行前に必ずコードを確認して安全性を確かめる。
  * **スター数を参考に**: 多くのスターがついているスキルは、他のユーザーによって検証されている可能性が高い。
* **陥りやすい罠 (Antipatterns)**:
  * **盲目的なインストール**: 中身を確認せずにスキルをインストールし、予期せぬ挙動やセキュリティリスクを招く。
  * **過剰なスキルの追加**: 必要以上のスキルを追加し、エージェントのコンテキストを圧迫する。

## **17. 総評**

* **総合的な評価**:
  SkillsMPは、170万件以上という膨大なスキルを網羅し、急速に拡大するAIエージェントエコシステムにおいて、ユーザーとスキルをつなぐ最大のハブとしての役割を果たしています。APIを通じた連携などシステムへの組み込みやすさも兼ね備えており、非常に価値が高いプラットフォームです。
* **推奨されるチームやプロジェクト**:
  Claude CodeやOpenAI Codex CLIを積極的に活用し、業務や職務に特化した機能を素早く追加して開発効率を上げたいすべてのプロフェッショナル。
* **選択時のポイント**:
  SaaS連携などの堅牢な認証統合が必要な場合はComposioなどを検討すべきですが、開発支援や自律的AIワークフローに膨大な公開知識（スキル）を手軽に連携させたい場合はSkillsMPが最適です。
